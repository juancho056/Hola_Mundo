    Thisislegal.com - Realistic Challenge 2

    Enunciado:

    You have heard about people being targeted by a new religion called Egitology. Another hacker infiltrated the group and discovered that the list of people they target is stored on the site but he doesn't know where.

    Pasos para resolver el reto:

    1. Se ingresa al reto, aparecen varios links, entre ellos logion.php, se intenta loguear con User: Admin y se aplica SQLI Inyeccion Password = 'or 1=1--'; lo que permite ingresar como Admin.

    2. Llama la atención al ingresar que la cuenta es limitada y el Link Back up Data Base. Se ingresa a este linkmuestra varios datos entre ellos (1, Admin, '3819585098e8cfe6c4fe4fcfd3a0fd59' ,'Limited' y (2, 'SuperAdmin', '8cc4ba204dd44cc92d7646ad035b7647', 'Unlimited') .

    3. Contando los caracteres se establece que es criptografía MD5. Admin= xenu5 ; SuperAdmin= lmon12

    4. Se sale por completo se vuelve a ingresar pero esta vez como Username=SuperAdmin; Password= lmon12;

    Aparecen todos los archivos.php que se visualizaban inicialmente en la pagina y dos carpetas adicionales

    • Images
    • misc

    Para solucionar el reto se deben borrar las siguientes carpetas:

    • /root/Misc/targets que es el objetivo
      • /root/images/logs para claear los registros y no dejar evidencia alguna.

    Una vez se borran ambas carpetas el reto esta resuelto.
